# Sistema de Compras

Este é um projeto desenvolvido em **Java** com o intuito de praticar conceitos de listas, coleções de dados e métodos de ordenção, além de reforçar a programação orientada a objetos.

---

## Sobre o Projeto

O **Sistema de Compras** é uma aplicação que simula o uso de um cartão de crédito, permitindo registrar compras e controlar o saldo disponível com base em um limite definido.

O objetivo do projeto é praticar conceitos fundamentais da linguagem Java, incluindo:

- Manipulação de listas com `ArrayList`
- Uso da interface `List`
- Implementação de métodos de ordenação de listas por meio da interface `Comparable`.
- Estruturas de repetição e decisão
- Programação orientada a objetos (POO)
- Organização de código em classes
- Trabalhar com objetos e suas hierarquias

---

## Funcionalidades

O sistema permite:

- Definir um limite inicial para o cartão
- Registrar compras
- Armazenar compras em uma lista
- Atualizar o saldo disponível automaticamente
- Impedir compras quando o saldo for insuficiente
- Exibir a lista de compras realizadas
- Ordenar as compras
- Encerrar o programa conforme escolha do usuário

---

## Tecnologias Utilizadas

- Java
- `ArrayList`
- Interface `List`
- Interface `Comparable`
- Entrada de dados com `Scanner`
- Estruturas de controle (`while`, `if/else`)

---

## Estrutura do Projeto

O projeto é organizado em classes que representam os principais elementos do sistema, como:

- `CartaoDeCredito`
- `Compra`
- Classe principal (`Main`)

Cada classe possui seus próprios atributos e comportamentos, aplicando conceitos de organização, reutilização de código e manipulação de listas.

---

## Como Executar o Projeto

1. Clone o repositório:

`git clone https://github.com/LucasSilveira30/Sistema-Compras.git`

2. Acesse a pasta do projeto:

`cd Sistema-Compras`

3. Compile os arquivos Java:

`javac *.java`

4. Execute a aplicação:

`java Main`
